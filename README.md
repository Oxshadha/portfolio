# Personal Portfolio Website

A modern, responsive personal portfolio website designed to showcase projects, technical skills, and experience as an AI & Machine Learning student.

## 🔗 Live Preview

[View Live Site](https://oxshadha.github.io/portfolio/)

## ✨ Features

- **Responsive Design**: Fully responsive layout optimized for desktop, tablet, and mobile devices.
- **Modern UI/UX**: Clean, premium aesthetic with smooth micro-animations and interactive hover effects.
- **Dynamic Elements**: Custom mathematics-themed loading animations and floating background symbols.
- **Interactive Navigation**: Sticky navigation bar with smooth scrolling and active section highlighting.
- **Contact Form**: Fully functional contact form integrated with FormSubmit.

## 🛠️ Built With

- **HTML5**: Semantic and accessible markup.
- **CSS3**: Custom styling using CSS variables, Flexbox/Grid layouts, and keyframe animations (No external CSS frameworks used).
- **Vanilla JavaScript**: DOM manipulation, scroll event handling, and custom loader logic.
- **Google Fonts**: Premium typography using 'Poppins'.
- **Material Icons**: Scalable vector icons for UI elements.

## 📁 Repository Structure

```text
portfolio/
├── index.html          # Main HTML structure
├── styles.css          # Core stylesheet and animations
├── script.js           # Interactive functionality
├── Profile pictures/   # Assets for profile imagery
└── Achivements/        # Assets for achievements section
```
